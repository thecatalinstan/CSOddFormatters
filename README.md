## Getting Started

### Installation through CocoaPods

Install using [CocoaPods](http://cocoapods.org) by adding this line to your Podfile:

````ruby
use_frameworks!
pod ‘CSOddFormatters’
````

### In your Project

```swift
import CSOddFormatters

print("\(CSOddFormatters.sharedHelper().systemInfoString)"
```
